# Complete CMake and Make Training Program

This comprehensive training program provides hands-on experience with CMake and Make build systems through practical projects and exercises.

## Training Structure

### Phase 1: Foundations (Week 1)
- Basic CMake projects
- Simple Makefiles
- Understanding build concepts

### Phase 2: Intermediate Skills (Week 2)
- Multi-file projects
- Libraries and dependencies
- Testing integration

### Phase 3: Advanced Techniques (Week 3)
- Cross-platform development
- Plugin architectures
- Custom modules

### Phase 4: Integration & Deployment (Week 4)
- CI/CD integration
- Package distribution
- Real-world scenarios

## Hands-On Projects

### Basic Projects
1. **Hello World**: Simple CMake executable
2. **Library Creation**: Building and using libraries
3. **Multi-File**: Managing multiple source files
4. **Pure Make**: Traditional Makefile usage

### Intermediate Projects
1. **Calculator**: Multi-file application with tests
2. **Logger**: Thread-safe library with file I/O
3. **Data Processor**: Configuration-driven processing
4. **Network Client**: External dependencies

### Advanced Projects
1. **Plugin Architecture**: Dynamic loading system
2. **Cross-Platform Lib**: Multi-OS compatibility
3. **CMake Package**: Distributable packages
4. **Performance Benchmark**: Optimization techniques

Before starting this training:
- Basic knowledge of C/C++ programming

## Getting Started

### Day 1: Basic CMake
1. Navigate to `training_projects/basic/hello_world`
2. Follow the build instructions
3. Modify the code to understand concepts
4. Complete the exercises

### Day 2: Libraries and Dependencies
1. Work through `training_projects/basic/library_example`
2. Understand the difference between PUBLIC, PRIVATE, and INTERFACE
3. Practice linking libraries

### Day 3: Multi-File Projects
1. Complete `training_projects/basic/multi_file`
2. Learn about include directories and target management
3. Practice with multiple source files

### Day 4: Pure Makefiles
1. Work with `training_projects/basic/simple_make`
2. Understand Makefile syntax and patterns
3. Learn about automatic dependencies

### Continue with Intermediate and Advanced Projects
